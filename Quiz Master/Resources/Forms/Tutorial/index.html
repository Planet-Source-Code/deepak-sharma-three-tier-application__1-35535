INTRODUCTION

This is a complete example of 3 Tier application.Before i tell you about this application i would like to clear you about the 3 Tier application.

Generally there are 3 types of Tier applications.

2 Tier application :   When we use normal type database programming eg:- (storing the record through VB to any database) that is called client/server programming in which VB is client and Server is any database. In this type of application VB (client) is 1 Tier and database (server) is 2 Tier.Client directly intract with the server and send his request directly to server. All the validation exists in the client application.

3 Tier application :  These kind of application are advance type client/server application.In 3 Tier application client is not intract with the database but the third layer intract with the database.

In 2 Tier application there are 2 layers

  • 1) User Layer (Client - VB)
  • 2) Data Layer (Database - Any)
In 3 Tier application there are 3 layers

  • User Layer (Client - VB)
  • Business Layer (Middleware - COM/COM+)
  • Data Layer (Database - Any)
3 Tier application can be made with COM technolgy.In these type of application all the database part eg:- adding updaing,deleteing, fetching,etc plus business related data eg:- person contact,name,fax etc are inside the bussiness layer . This business layer application intract with the database and client intract with the business layer . there is no database coding in the client side. One advantage of three Tier of application is this that you can reuse that business layer in any other programming language eg:- ASP,VC++,C++ etc. In this quiz application example i have use this com (business layer) in ASP also.

N Tier application :   In this kind of application there are lot of middleware between user and data layer.
